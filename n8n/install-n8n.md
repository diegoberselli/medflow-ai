# Instalação N8N

## 1. Instalar Node.js
- Baixar: https://nodejs.org/
- Versão LTS recomendada

## 2. Instalar N8N
```bash
npm install n8n -g
```

## 3. Iniciar N8N
```bash
n8n start
```

## 4. Acessar Interface
- URL: http://localhost:5678
- Criar conta inicial

## 5. Importar Workflow
- Usar arquivo: medical-workflow.json
